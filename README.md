## 文件说明

- **ESP32.ino** : 控制舵机的代码  
- **ASRPRO.hd** : 语音唤醒的代码  
- **3D Clock Slice.3mf** : 3D 打印时钟片文件  
- **Clock Dial CAD.dxf** : 亚克力背景板 CAD
